# 🚀 SOS Dépannage France - Version 1.0

![SOS Dépannage Banner](https://images.unsplash.com/photo-1621905251189-08b45d6a269e?auto=format&fit=crop&q=80&w=1200)

## ✨ Le Concept : L'excellence du service à domicile
**SOS Dépannage France** est une plateforme premium de mise en relation ultra-rapide entre clients et artisans certifiés. Conçue pour l'urgence et la qualité, l'application offre une expérience utilisateur fluide et sécurisée.

### 🛠️ Nos Domaines d'Expertise
- **💧 Plomberie** : Dépannage fuites, installation sanitaire.
- **🔐 Serrurerie** : Ouverture fine, blindage de porte.
- **⚡ Électricité** : Mise aux normes, bornes de recharge.
- **🔥 Chauffage** : Entretien chaudière, thermostats intelligents.
- **🚿 Assainissement** : Curage et débouchage haute pression.

---

## 💎 Fonctionnalités Premium
- **Interface High-Contrast** : Design moderne inspiré des standards SaaS de luxe.
- **Dashboard Admin** : Console de gestion complète pour les commandes et interventions.
- **Espace Client** : Suivi en temps réel et historique des prestations.
- **Multi-langue** : Support complet Français (FR) et Anglais (EN).
- **Responsive Design** : Expérience parfaite sur mobile, tablette et desktop.

---

## 🚀 Stack Technique
- **Frontend** : React 19, Tailwind CSS, Framer Motion, Lucide Icons, Shadcn/UI.
- **Backend** : Python FastAPI, MongoDB (prêt pour la production).
- **Déploiement** : Prêt pour Vercel & GitHub Actions.

---

## 📦 Installation (Local)

### Frontend
```bash
cd frontend
npm install --legacy-peer-deps
npm start
```

### Backend
```bash
cd backend
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python server.py
```
